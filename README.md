## Описание программы
Training Planner - это десктопное приложение с графическим интерфейсом для планирования и отслеживания тренировок. Программа позволяет вести журнал тренировок, фильтровать записи по типу и дате, а также просматривать подробную статистику.

### Основные возможности:
- Добавление тренировок с указанием даты, типа и длительности
- Автоматическая проверка корректности ввода (дата в формате ГГГГ-ММ-ДД, длительность > 0 и ≤ 1440 минут)
- Фильтрация тренировок по типу и диапазону дат
- Подробная статистика: общее количество тренировок, общая/средняя длительность, статистика по каждому типу
- Сохранение и загрузка данных в формате JSON
- Удобный табличный интерфейс на основе tkinter
